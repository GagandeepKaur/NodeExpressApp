NodeExpressApp

To run the application on your own:

  1. Install Mongodb

  2. Start Mongodb by going to bin folder and type mongod.exe --dbpath path to data folder in NodeExpressApp

  3. Run node server.js

  4. Run http://localhost:3000/ in your explorer to see the app.
